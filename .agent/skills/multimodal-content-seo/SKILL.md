# Multimodal Content SEO Skill

## When to use this skill
- When publishing activity photos/videos
- When creating tutorials or guides
- When optimizing media assets

## How to use this skill
1. **Image Alt & Semantic Labels**  
   Use AI-generated descriptive alt texts for images.  
2. **Video Transcripts**  
   Include text transcripts with keywords and entities.  
3. **Visual Entity Signals**  
   Tag images with rich metadata to help AI vision systems interpret them.  
4. **Voice Search Optimization**  
   Use conversational answers and audio tags.
