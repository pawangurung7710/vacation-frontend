# GEO Structured Data Skill

## When to use this skill
- On package pages (EBC, ABC, Poon Hill)
- On “About” and author bio pages
- On FAQ and review sections
- On homepage and contact page

## How to use this skill
1. **Generate Organization Schema**  
   Include accurate NAP (Name, Address, Phone), logo, and social profiles.  
2. **Person Schema**  
   Add author/guide credentials with bio links and real experience data.  
3. **TouristTrip / TravelPackage Schema**  
   Define each trekking package with itinerary, duration, price, and related entities.  
4. **FAQ & Review Schema**  
   Mark up FAQ sections and customer testimonials to enable rich AI/AEO features.
5. **Validation**  
   Test with Google Rich Results and Schema Markup Validator to ensure compliance.
