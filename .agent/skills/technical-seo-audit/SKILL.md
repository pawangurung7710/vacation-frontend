# Technical SEO Audit Skill

## When to use this skill
- After site launch
- After major content updates
- When speed or traffic issues appear

## How to use this skill
1. **Core Web Vitals**  
   Monitor LCP, INP, CLS and fix issues (image optimization, lazy loading).  
2. **Accessibility & Semantic HTML**  
   Use clear tag structures and ensure screen readers and AI bots can interpret content.  
3. **Crawlability**  
   Review robots.txt, sitemap.xml, and ensure key pages are indexable.  
4. **Mobile-First Optimization**  
   Validate mobile navigation, load speed, and responsive design.
