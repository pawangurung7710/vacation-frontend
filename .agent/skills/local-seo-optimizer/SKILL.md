# Local SEO Optimizer Skill

## When to use this skill
- When setting up Google Business Profile
- When targeting location-based keywords
- When optimizing for local discovery

## How to use this skill
1. **Google Business Profile Setup**  
   Complete NAP, categories, photos, and reviews.  
2. **Local Citations**  
   Submit to Nepal travel directories, tourism boards, and niche sites.  
3. **Location Page Creation**  
   Create dedicated pages for “Trekking in Kathmandu” etc., with structured content.  
4. **Review Strategy**  
   Prompt customers for reviews after service; respond professionally.
