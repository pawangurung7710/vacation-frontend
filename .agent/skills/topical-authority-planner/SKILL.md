# Topical Authority Planner Skill

## When to use this skill
- For large topic areas (Everest, Annapurna, Travel Guides)
- When expanding content breadth
- When internal linking strategy is needed

## How to use this skill
1. **Define Pillar Topics**  
   Create anchor pages covering broad themes (e.g., “Complete Everest Base Camp Guide”).  
2. **Identify Cluster Subtopics**  
   List supportive nodes (season guide, packing list, costs, safety tips).  
3. **Internal Linking Plan**  
   Ensure clusters link back to pillar pages with contextual anchor text.  
4. **Publication Calendar**  
   Schedule and interlink content strategically over time.
