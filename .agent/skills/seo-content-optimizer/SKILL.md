# SEO Content Optimizer

## When to use this skill
- Creating or updating website pages (packages, guides, blog posts)
- Rewriting content to target specific user and AI search intent
- Improving existing pages to increase visibility in AI summaries

## How to use this skill
1. **Answer-First Introduction**  
   Always start content with a concise answer that directly responds to search intent within first 2–3 sentences.  
2. **Topic & Intent Mapping**  
   Identify primary user intent (informational, commercial, or transactional) before drafting the outline.  
3. **Entity-Rich NLP**  
   Include relevant entities (e.g., trek names, locations, guide experience references) rather than isolated keywords.  
4. **Chunkable Structure**  
   Break content into short sections with clear headings, bullet lists, and comparison tables for easier parsing by AI and humans.  
5. **Internal Linking**  
   Link related topics (e.g., season, safety, permit guides) to enhance semantic relevance across pages.
