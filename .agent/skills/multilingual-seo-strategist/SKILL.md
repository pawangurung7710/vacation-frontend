# Multilingual SEO Strategist Skill

## When to use this skill
- After English SEO is fully established and generating consistent leads.
- When expanding into specific markets like Germany, France, Spain, or China.
- On core money pages (EBC, ABC, Poon Hill) for specific language subdirectories.

## How to use this skill
1. **Hreflang Implementation**
   Ensure correct technical setup of hreflang tags for different language variations (e.g., /de/, /es/, /fr/).
2. **Cultural Localization**
   Do not just translate; adapt the tone, formatting, and cultural expectations for the target market. (e.g., German markets often require extreme detail on safety and daily itineraries).
3. **Localized Keyword Research**
   Identify search volume and intent for native language queries (e.g., "Everest Base Camp Trek" vs. "Wandern im Everest-Gebiet").
4. **Local Link Building**
   Acquire backlinks from native-language travel blogs and local domains (.de, .fr, .es) to signal relevance to local search engines.
