# Workspace Rules – frontend Development

# GLOBAL WEBSITE RULES
Project: Vacation Nepal Trekking
Positioning: Premium, international Himalayan trekking brand

This document defines global rules that apply to ALL pages.

------------------------------------------------------------
1. BRAND POSITIONING
------------------------------------------------------------

- Premium but authentic
- Local Nepali expertise with global trust standards
- Clean, modern, international feel
- Similar quality level to major travel brands (not local directory style)

Avoid:
- Cluttered layouts
- Overuse of colors
- Overcrowded content blocks
- Aggressive promotional design

------------------------------------------------------------
2. DESIGN SYSTEM (Himalayan Serenity Palette)
------------------------------------------------------------

Background: #F8FAFC
Primary Text / Headings: #2C4A6A
CTA Accent: #006D77
Earth Accent (icons/dividers): #5D4037
White: #FFFFFF
Borders: #E0E0E0
Dark text: #333333

------------------------------------------------------------
3. VISUAL STYLE RULES
------------------------------------------------------------

- Large typography for headings
- Generous white space
- Rounded corners (rounded-xl or rounded-2xl)
- Soft shadows (shadow-sm, shadow-md only on hover)
- Clean grid spacing (gap-8 minimum)

Images:
- Use high-quality real trekking images
- Avoid generic stock photos
- Use 16:9 ratio for cards
- Always use optimized responsive images

------------------------------------------------------------
4. UX PRINCIPLES
------------------------------------------------------------

- Clear visual hierarchy:
  Title > Stats > Price > CTA

- Do not overwhelm users
- Keep interface breathable
- Mobile-first responsive design
- Sticky navigation must remain consistent across pages
- Footer must remain identical across all pages

------------------------------------------------------------
5. SEO RULES (CRITICAL)
------------------------------------------------------------

Each page must include:
- Proper H1
- Structured heading hierarchy (H2, H3)
- Meta title and description placeholder
- Internal linking to relevant pages
- Bottom SEO content block (where relevant)

Use natural language. No keyword stuffing.

------------------------------------------------------------
6. COMPONENT RULES
------------------------------------------------------------

- Components must be reusable
- No hardcoded repeated UI
- No duplication of homepage sections on inner pages
- Avoid repeating "Why Travel With Us" fully on other pages
- Keep animations subtle and premium

------------------------------------------------------------
7. INTERACTION DESIGN
------------------------------------------------------------

Hover Effects:
- Slight card lift
- Image zoom 2–3%
- Smooth 300ms transition
- CTA fills on hover

Never use flashy or dramatic animations.

------------------------------------------------------------
8. TRUST & INTERNATIONAL APPEAL
------------------------------------------------------------

- Always show USD pricing clearly
- Avoid informal tone
- Maintain global credibility tone
- Use clean professional iconography

------------------------------------------------------------

9. Check the /.ai/project folder for other detail rules
END OF GLOBAL RULES
