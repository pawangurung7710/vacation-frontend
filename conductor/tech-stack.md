# Tech Stack: Vacation Nepal Trekking

## Core Framework
- **Framework:** Next.js 14+ (App Router)
- **Language:** TypeScript
- **Rendering:** SSR + SSG (SEO-first approach)

## Styling & UI
- **Styling:** Tailwind CSS (with `tailwind-merge` and `clsx`)
- **Icons:** Lucide React
- **Design:** Spacious, clean typography, Himalayan color palette

## State & Forms
- **Data Fetching:** TanStack Query (React Query)
- **Local State:** Zustand (if needed)
- **Forms:** React Hook Form + Zod (Validation)

## Rich Media & Visuals
- **Animations:** Framer Motion (Subtle & Premium)
- **Charts:** Recharts (Altitude profiles)
- **Maps:** Mapbox GL JS (Route visualization)
- **Images:** Next.js Image Component (WebP, Responsive)

## SEO & Deployment
- **SEO:** Next.js Metadata API, Open Graph, JSON-LD Structured Data
- **Deployment:** Vercel (Frontend)
