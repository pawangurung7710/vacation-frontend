Follow these strict rules:

1. Design must feel international, not local.
2. Use spacious layout and clean typography.
3. Avoid clutter.

5. Always prioritize readability.
6. Follow the defined design system.
7. All components must be modular.
8. Code must be scalable and production-ready.
9. Optimize for SEO and performance.
10. Keep animations subtle and premium.
